JavaScript Basics: A Beginner's Guide

1. Variables

Variables are used to store data values in JavaScript. You can define variables using var, let, or const.

Example:

let x = 10;

2. Functions

Functions are blocks of code that can be reused. You can declare a function with the function keyword.

Example:

function greet() {
   console.log('Hello, World!');
}

3. Loops

Loops allow you to run the same block of code repeatedly. Common loops include for, while, and do...while.

Example:

for (let i = 0; i < 5; i++) {
   console.log(i);
}
